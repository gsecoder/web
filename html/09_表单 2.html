表单的作用：

1.表单用于搜集不同类型的用户输入（搜集用户信息） 2.用户提交表单时向服务器传输数据，从而实现用户与 Web 服务器的交互 3.表单是一个包含表单元素的区域 4.表单使用表单标签（form）定义

from表单的常见属性：

1. action=“URL”：一个 URL 地址；指定 form 表单向何处发送数 2. name 作用是给表单起名，为了便于我们 3. enctype =“string”：规定在发送表单数据之前，如何对表单数据进行编码(multipart/form-data"，以二进制传输) 4. method =“get/post”：指定表单以何种方式发送到指定的服务器程

一个完整的表单包含三个基本组成部：表单标签、表单域、表单按钮

1.表单标签：是指form标签本身，它是一个包含表单元素的区域，使用form /form 2.表单域：是form标签中用来收集用户输入的每一项，通常用 input 标签来定义，input 标签有不同的类型，对应用户不同的数据 3.表单按钮：用来提交from表单中的所有信息到服务器
